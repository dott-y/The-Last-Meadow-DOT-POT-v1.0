## 🚀 How to use

### Method 1: Console Copy-Paste (Easiest & Recommended)

1. Open **The Last Meadow** game in your browser
2. Press **F12** or **Ctrl+Shift+I** to open Developer Tools
3. Click the **Console** tab
4. **Copy the entire script** from below:
   - Go to: https://raw.githubusercontent.com/dott-y/The-Last-Meadow-DOT-POT-v1.0/refs/heads/main/dot.js
   - Press `Ctrl+A` (Select All)
   - Press `Ctrl+C` (Copy)
5. **Paste** into the console (`Ctrl+V`)
6. Press **Enter**

✅ **Done! The bot will start automatically!**
